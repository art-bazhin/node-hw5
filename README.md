# Node.js: серверный JavaScript
## Домашняя работа №5

### Использованные технологии

* Koa.js
* MongoDB + Mongoose

### Установка

```
npm i
```

### Запуск

```
npm run start
```

### Разработка

```
npm run dev
```

Для входа в профиль администратора: логин - admin, пароль - password.

[Приложение на heroku.com](https://node-hw5.herokuapp.com/)
